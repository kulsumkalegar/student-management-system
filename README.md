# Student Management System

A CLI-based Student Management System built in Go.

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- JSON File Storage

## Technologies Used

- Go
- JSON
- File Handling
- Packages
- Git & GitHub

## Project Structure

```
student-management-system/
│
├── main.go
├── go.mod
├── students.json
│
├── models/
├── services/
└── storage/
```
